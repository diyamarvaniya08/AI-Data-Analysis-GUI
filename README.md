# AI-Data-Analysis-GUI
A professional desktop data analysis software built using Python, Pandas, Tkinter, and Matplotlib.
This tool allows non-technical users to analyze Excel or CSV files, generate reports, and create charts without writing any code.

# Features
 - It provides a clean User-friendly graphical interface where users can:
     - Upload CSV or Excel files
     - Automatic column detection
     - Group-by analysis
     - Multiple aggregation options
     - Chart generation (Bar, Line, Pie, Column)
     - Interactive report & chart preview
     - Export reports (Excel/CSV)
     - Export charts as PNG
